# How to Contribute (Simple Guide for Beginners)

## Easy Way: Edit Directly on GitHub (No Git Commands!)

### Step 1: Find the file you want to edit
1. Go to the repository on GitHub
2. Navigate to the file (e.g., `public/index.html`)
3. Click the **pencil icon** (‚úèÔ∏è) to edit

### Step 2: Make your changes
1. Edit the file in the browser
2. Preview if possible

### Step 3: Save your changes
1. Scroll down to "Propose changes"
2. Write a short description (e.g., "Updated event information")
3. Click **"Propose changes"**
4. Click **"Create pull request"**
5. Done! Wait for approval

## Alternative: Using GitHub Desktop (Easier than Command Line)

### First Time Setup
1. Download **GitHub Desktop** from https://desktop.github.com
2. Install and sign in with your GitHub account
3. Click **File ‚Üí Clone Repository**
4. Select `ictclub-website` and choose where to save it

### Making Changes
1. Open GitHub Desktop
2. Click **Current Branch ‚Üí New Branch**
3. Name it (e.g., "update-events")
4. Click **Create Branch**
5. Open the files in your favorite editor (VS Code, Notepad++, etc.)
6. Make your changes and save

### Submitting Your Work
1. Go back to GitHub Desktop
2. You'll see your changes listed
3. Write a summary (e.g., "Added new event")
4. Click **Commit to update-events**
5. Click **Publish branch** (or **Push origin**)
6. Click **Create Pull Request**
7. Fill in the form and submit
8. Done! Wait for review

## For Command Line Users

```bash
# 1. Get the code
git clone https://github.com/mwecauictclub/ictclub-website.git
cd ictclub-website

# 2. Create your branch
git checkout -b your-branch-name

# 3. Make changes to files

# 4. Save your changes
git add .
git commit -m "Description of what you did"

# 5. Upload
git push origin your-branch-name

# 6. Go to GitHub and create Pull Request
```
